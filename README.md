# 🛡️ VAPS — Digital Babylon Finance
**Проект группы DS-24Б: Жаркеев Ерден, Еркен Алмаз**

### 🎯 О проекте
VAPS — это интеллектуальный сервис, который превращает хаос в кошельке в четкий капитал. 
Мы объединили принципы книги "Самый богатый человек в Вавилоне" и мощь анализа данных на Python.

### 🛠 Стек технологий
* **Python** (Core logic)
* **Streamlit** (Web Interface)
* **Plotly** (Interactive Visualizations)
* **Pandas** (Data Processing)

### 🚀 Функционал
- Мониторинг "утечек" бюджета.
- Автоматический расчет правила "1/10" (Плати себе первым).
- Интерактивные чарты расходов.
- Система микро-обучения.
